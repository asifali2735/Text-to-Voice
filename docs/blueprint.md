# **App Name**: Jannu Connect

## Core Features:

- User Feed: Display a personalized feed of short-form videos and posts from followed users and recommended content.
- Content Creation: Allow users to easily record, edit, and share short videos with filters, effects, and background music. The videos should have a watermark containing the app logo.
- Content Discovery: Implement a recommendation system to suggest relevant content based on user activity and interests. A tool powered by a generative AI should determine when to show specific types of sponsored content.
- Social Interaction: Enable users to follow each other, like, comment on, and share content within the app.
- Trending Topics: Identify and display trending topics and hashtags to encourage participation in popular conversations.
- Push Notifications: Send push notifications to users for new followers, likes, comments, and other activity.

## Style Guidelines:

- Primary color: Saturated pink (#E91E63) for a vibrant, social feel.
- Background color: Very light pink (#FCE4EC).
- Accent color: Purple (#9C27B0), for analogous contrast.
- Font pairing: 'Poppins' (sans-serif) for headlines, 'PT Sans' (sans-serif) for body.
- Use simple, line-based icons for a modern and clean look.
- A user-friendly layout for a seamless content discovery experience, featuring a prominent video player and a clear comment section.
- Smooth transitions and animations to enhance the user experience, for example on swiping between videos.
